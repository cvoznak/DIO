Estrutura de repetição

- O real poder dos computadores está na sua habilidade para repetir uma operação ou uma série de operações várias vezes.
- Cada repetição é chamada laço (loop) é um dos conceitos básicos da programação estruturada.
- Uma estrutura de repetiçãao permite que uma sequência de comandos seja executada repetidamente, caso determinadas condições sejam satisfeitas.
- Essas condições são representadas por expressões lógicas.

Classificação:

- Repetição com teste no início (while)
- Repetição com teste no final (do-while)
- Repetição contada (for)

Break

- O comando break é utilizado para terminar de forma abrupta uma repetição
- Pode ser utilizado fora de uma estrura de reptição.

Continue

- Quando o comando continue é executado, os comandos restantes são ignorados e o programa volta a testar novamente ou não.
- Não pode ser utilizado fora de uma estrutura de repetição.
- Muito pouco utilizado.

While

- Se retornar TRUE executa a sentença (enquanto verdadeiro ou diferente de false)
- Break pula para fora da repetição, continue, mantem na repetição.

do-while

- Executa a sentença uma primeira vez, independente de true or false.
- Enquanto true mantem a repetição.
- Se false após a primeira execução pula para fora da repetição.

for

- Executa por um numero limite de repetições, através de uma atualização após cada execução.

Operadores de incremento e decremento

- Pré-fixados:
	++numero 	-> numero = numero + 1	-> retorne numero
	--numero	-> numero = numero - 1	-> retorne numero

- Pós-fixados:
	numero++	-> retorne numero	-> numero = numero + 1
`	numero--	-> retorne numeor	-> numero = numero - 1


Arrays

- É um objeto utilizado para armazenar sequencialmente dados do mesmo tipo.
- Permanecem com o memso tamanho depoiss de criados

Tipos:

- Array Unidimensional (vetor)
	a[1] -> a nome do array / [1] indice ou subscrito do elemento no array a.
- Array Multimensional (matriz)
	b[0][3] -> array b / [0] linha 0 / [3] coluna 3
